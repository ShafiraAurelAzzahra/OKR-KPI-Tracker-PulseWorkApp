# OKR-KPI-Tracker-PulseWorkApp
Sistem Manajemen Kinerja berbasis cloud modern yang dirancang untuk membantu organisasi melacak Objectives and Key Results (OKR) serta Key Performance Indicators (KPI) karyawan secara real-time. Aplikasi ini menggunakan Google Apps Script (GAS) sebagai backend serverless dan Google Spreadsheet sebagai basis data utama.
