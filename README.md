# 🚀 PulseWork

Sistem Manajemen Kinerja berbasis cloud yang dirancang untuk membantu perusahaan, startup, UMKM, maupun organisasi dalam memantau Objectives and Key Results (OKR) serta Key Performance Indicators (KPI) karyawan secara real-time.

PulseWork dibangun menggunakan Google Apps Script sebagai backend serverless dan Google Spreadsheet sebagai database utama sehingga dapat digunakan tanpa memerlukan server hosting tambahan.

---

## 📋 Tentang PulseWork

PulseWork adalah platform OKR & KPI Tracker modern yang membantu perusahaan memonitor performa tim, pencapaian target, dan perkembangan karyawan dalam satu dashboard terintegrasi.

Terinspirasi dari platform seperti ClickUp, Asana, Monday.com, dan Jira, PulseWork menghadirkan pengalaman penggunaan yang cepat, responsif, dan mudah digunakan.

### 🎯 Visi

Membantu organisasi membangun budaya kerja yang berorientasi pada tujuan, transparansi, dan peningkatan performa berkelanjutan.

### 🚀 Misi

- Mempermudah monitoring OKR dan KPI.
- Menyediakan dashboard performa secara real-time.
- Membantu manajemen mengambil keputusan berbasis data.
- Meningkatkan produktivitas individu maupun tim.
